# Reference Agent

Python implementation (built on Google's Agent Development Kit) that produces and visualizes OKF bundles: it drafts OKF documents from a BigQuery source, enriches them with web-crawled citations, and ships a `viewer/generator.py` that renders the bundle as a self-contained, interactive graph.
